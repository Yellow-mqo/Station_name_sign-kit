# 新陽社製　薄型電気掲示器(SE型) KIT

JR東日本管内などでよく見かける形の駅名標を作成するためのKITです。  
同じ形の物であれば、用意するテクスチャ次第で駅名標以外でも番線案内や、乗換案内なども作成可能です。  
なお駅名標であれば、"[Sigene駅名標ジェネレーター](https://tr246.github.io/Sigene/)"の使用を想定･推奨としています。  
  
## 🛠 内容物
- モデルファイル(SE型のサイズ差分7×2種)
- SE-6型用の表示部を含むテクスチャ
- 仮jsonファイル(1つ)
- Readme及び説明用テキストファイルそれぞれ一つ  
  ※フォルダ構成済み
  
## 🎮 実装について
デフォルトで照明モデルとして読み込みます。  
設置の際、吊り下げ型なので天井(ブロックの下面)に向かって設置するだけです。  
(もちろんモデルから支柱を削除し、位置を調整すれば壁掛けタイプも作成可)
